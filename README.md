# Airport Management System Database

A database project for designing and implementing an Airport Management System using SQL and EER modeling.

## Overview
This system manages airport operations such as flights, passengers, airlines, staff, gates, terminals, baggage tracking, and boarding passes.

## Project Components
- Conceptual Model (EER Diagram)
- Relational Model Design
- SQL Database Implementation
- Sample Data Insertion
- SQL Queries and Results

## Tools Used
- SQL
- Database Design
- EER Modeling
- Oracle SQL Developer
